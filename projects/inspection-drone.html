// 01 · System Overview

Autonomous Structural Inspection Pipeline

This project developed a fully autonomous drone system capable of inspecting structures such as bridges and aircraft fuselages using aerial 3D point cloud processing. The drone scans the target structure in flight, reconstructs a dense 3D model, and detects surface anomalies — all without relying on GPS or pre-placed markers.

The system was validated in Gazebo simulation with a realistic aircraft fuselage model, demonstrating the complete scan-to-reconstruction pipeline in a controlled environment before field deployment.

Full inspection mission — autonomous drone scanning aircraft fuselage structure and generating 3D point cloud.

// 02 · Tech Point 1

SIFT Keypoint Extraction

Scale-Invariant Feature Transform (SIFT) is applied to each camera frame to extract robust keypoints that are invariant to scale, rotation, and illumination changes. These keypoints serve as the foundation for all subsequent 3D reconstruction and pose estimation steps — enabling reliable feature detection even on uniform, textureless structural surfaces.

SIFT keypoint extraction — scale and rotation invariant feature detection on aircraft fuselage surface.

Tech Stack SIFT (OpenCV) · Feature Descriptor Computation · Keypoint Filtering · ROS

// 03 · Tech Point 2

Inter-Frame Keypoint Matching

SIFT descriptors extracted from consecutive frames are matched using FLANN-based matching with Lowe's ratio test to filter out ambiguous correspondences. Accurate inter-frame correspondences enable reliable estimation of relative camera motion, which is the cornerstone of visual odometry and 3D reconstruction.

Keypoint matching result — SIFT correspondences between two consecutive drone camera frames.

Tech Stack FLANN Matcher · Lowe's Ratio Test · Homography Estimation · RANSAC Outlier Rejection

// 04 · Tech Point 3

2D–3D Feature Mapping & Point Cloud Generation

Matched 2D keypoints are lifted to 3D space through triangulation, establishing a sparse 3D point cloud of the structure. The PnP (Perspective-n-Point) algorithm is then applied to refine the drone's 6-DOF pose relative to the reconstructed map — enabling accurate localization against the growing 3D model.

2D–3D mapping result — triangulated 3D points from matched SIFT keypoints, forming the structural point cloud.

Tech Stack Triangulation · PnP (OpenCV solvePnP) · Bundle Adjustment · Sparse Point Cloud · ROS tf

// 05 · System Architecture

Full System Architecture

The complete inspection pipeline integrates perception, localization, and reconstruction into a single ROS-based framework. Camera input is processed through the SIFT → Match → Triangulate → PnP chain in real time, continuously updating both the drone's pose estimate and the accumulated 3D point cloud of the structure being inspected.

Full system architecture — end-to-end inspection pipeline from camera input to 3D structural map.

// 06 · Simulation & Validation

Gazebo Simulation & Results

The full system was validated in Gazebo with a high-fidelity aircraft fuselage model. The drone autonomously flew a scan trajectory around the structure while building a real-time 3D point cloud. Simulation results confirmed the accuracy of the reconstruction pipeline and the robustness of SIFT-based visual odometry under varying lighting conditions.

Gazebo simulation setup — drone scanning aircraft fuselage in a controlled virtual environment.
