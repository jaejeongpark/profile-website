// 01 · System Overview

Precision Landing on Moving Maritime Platforms

This project developed an autonomous precision landing system for drones operating in GPS-denied maritime environments. The core challenge: landing reliably on a moving ship deck that pitches, rolls, and translates — with no GPS reference and in open-water conditions.

The system uses a downward-facing camera to detect the landing target on the ship, estimates the relative 6-DOF pose in real time, and feeds continuous landing commands to the PX4 flight controller via ROS offboard control — achieving centimeter-level landing accuracy on a dynamically moving platform.

// 03 · Tech Point 1

Ship Recognition — YOLO Detection

YOLO is deployed as the primary detection module to identify the target ship and its landing pad from aerial imagery. The model was fine-tuned on maritime landing platform data to handle challenging conditions: glare off the water surface, varying altitudes, and partial visibility of the landing marker.

The detected bounding box is passed to the PnP pose estimation module, providing the initial localization of the landing target relative to the drone's camera frame.

YOLO ship recognition — real-time detection and tracking of landing platform on moving vessel from drone camera.

Tech Stack YOLOv5 · Custom Dataset (Maritime Landing Platform) · OpenCV · ROS · Real-time Inference

// 04 · Tech Point 2

Visual Landing Target & PnP Pose Estimation

A custom visual landing target is mounted on the ship deck. Its known 3D geometry combined with detected 2D corner points enables the PnP (Perspective-n-Point) algorithm to recover the full 6-DOF relative pose — providing the drone's position and orientation relative to the landing pad in real time.

The recovered pose is used to compute velocity commands in offboard mode, continuously correcting the drone's trajectory to compensate for ship motion — achieving stable, accurate touchdown even as the platform moves beneath it.

Custom landing target — designed for reliable YOLO detection and accurate PnP 6-DOF pose recovery at varying altitudes.

Tech Stack PnP (OpenCV solvePnP) · Custom Visual Marker · 6-DOF Pose Recovery · PX4 Offboard Control · ROS · Velocity Command Interface
